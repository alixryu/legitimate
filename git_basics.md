# Git Basics

## First Time Git Setup
``` shell
$ git config --global user.name "Albino Squirrel"
$ git config --global user.email asquirrel@oberlin.edu
```

## Git Clone
``` shell
$ git clone https://github.com/heavenk21/legitimate.git
```

## Git Init
``` shell
$ mkdir legitimate
$ cd legitimate
$ git init
```
